## **Features**

5. **Progress Tracker:** Visual representation of the task completion status.

## **Usage**

1. **Adding Tasks:** Enter task details and click "Add Task" to add a new task.
2. **Updating Tasks:** Click on a task to edit its details.
3. **Deleting Tasks:** Click the trash icon next to a task to delete it.
4. **Clearing All Tasks:** Click the "Clear All Tasks" button to remove all tasks.
5. **Progress Tracker:** View the progress of your tasks in the progress tracker.
